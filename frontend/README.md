## Frontend Setup

This project includes a React-based frontend interface for interacting with the **Health Insights AI Assistant**.

---

### Prerequisites

Make sure the following are installed:

- Node.js **v20 or higher**
- npm

Check your versions:
- node -v
- npm -v


---

### Running the Frontend Locally

1. Navigate to the frontend folder:
cd frontend


2. Install dependencies:
npm install


3. Start the development server:
npm run dev


4. Open the application in your browser:
http://localhost:5173
